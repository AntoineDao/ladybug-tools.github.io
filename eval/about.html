Our Story

[place holder] Ladybug Tools started by Mostapha as a side project in mid 2012. The first version which was released as Ladybug on 13 January 2013 was a collection of 24 components for weather data visualization, radiation and sunlight hours analysis in Grasshopper. The main goal of the project was to integrate environmental analysis in a parameteric modeling environment.
